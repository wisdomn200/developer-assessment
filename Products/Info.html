To review the code and run the application, please follow these instructions: Clone the repository from the Github URL: https://github.com/[username]/[repository-name].git Open a terminal and navigate to the project's root directory. Run the command npm install to install the required dependencies. Create a .env file in the project's root directory and set the REACT_APP_GRAPHQL_ENDPOINT variable to the URL of your GraphQL endpoint. Run the command npm start to start the application. Open a web browser and navigate to http://localhost:3000/ to view the application. You should see the products page with the products displayed in cards. Each card should have an "Add to cart" button. Clicking on the "Add to cart" button should show a cart summary component as an overlay and add the product to the cart. The cart summary component should allow you to increment (+), decrement (-), and delete functionality on each cart item in the cart. You can review the code in the project's source files located in the src/ directory. Note: You may need to have a GraphQL API running on a specified endpoint and update the .env file to reflect the GraphQL endpoint URL.To review the code and run the application, please follow these instructions: Clone the repository from the Github URL: https://github.com/[username]/[repository-name].git Open a terminal and navigate to the project's root directory.
